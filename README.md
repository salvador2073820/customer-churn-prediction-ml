# Predicción de Churn de Clientes con Machine Learning

## Descripción
Este proyecto implementa un **modelo de Machine Learning** para predecir la **cancelación de clientes (churn)** utilizando datos históricos de telecomunicaciones.

El objetivo principal es:
- Clasificar clientes según su probabilidad de churn
- Identificar clientes de **alto riesgo**
- Apoyar decisiones de retención mediante análisis de datos

---

## ¿Qué hace el proyecto?
- Limpia y preprocesa datos reales de clientes
- Realiza análisis exploratorio de datos (EDA)
- Entrena modelos de clasificación supervisada
- Evalúa el desempeño de los modelos
- Calcula la probabilidad de churn por cliente
- Segmenta clientes en niveles de riesgo (bajo, medio, alto)

---

## Modelos utilizados
- Regresión Logística
- Random Forest Classifier (con balanceo de clases)
